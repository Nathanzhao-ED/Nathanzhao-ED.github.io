2.位运算相关

都是在二进制下进行运算

位运算符 功能 例子 应用
& 与运算 3(11)&2(10)=2(10) x&1 看x二进制的尾位是否是1,快速幂
| 或运算 3(11)|2(10)=3(11) x|1 相当于在尾位+1
^ 异或运算 3(11)^2(10)=1(01) x^1 把尾位0 -> 1, 1 -> 0
>> 右移操作 x>>1相当于给x/2
<< 左移操作 x<<2相当于让x乘以4
~ 按位取反 ~11 = 00 while(~scanf("%d",&n)){}

x>>1|1; x/2+1

x + y >> 1; (x + y) / 2

不用管直接加()搞定
